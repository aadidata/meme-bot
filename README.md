# MemeBot 🤖

A Discord bot built in Python that fetches memes from Reddit.

## Commands
- `!meme` - fetches a random meme of any genre and displays it 
- `!meme [subreddit]` - capable of fetching a meme of a specific genre/subreddit
- `!memeinfo` - gives info about the meme like title ,upvotes , author , which subreddit from 
- `!leaderboard` - shows the leaderboard of how many times someone has used '!meme"
- `!ping` -  replies with 'pong!'
- `!hello` - memebot replies with a hello

## Setup
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Add your Discord token to `.env`
4. Run: `python bot/main.py`
